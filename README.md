# Bike Sales Customer Analysis Dashboard

Tools: Excel · Pivot Tables · Dashboard · Conditional Formatting

## About
Analyzed 1,000+ customer records to identify which demographic profiles 
are most likely to purchase a bike.

## Key Insight
Middle-aged customers (31–54 yrs) with a 0–1 mile commute and 
$40K–$80K income showed the highest purchase conversion rate.

## Dataset
Source: Kaggle — Bike Buyers Dataset

## Skills Demonstrated
- Data Cleaning (IFERROR, TRIM, standardizing labels)
- Pivot Tables & Slicers
- Dashboard Design
- Conditional Formatting
